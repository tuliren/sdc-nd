Lane Line Detection
====

# Setup
```sh
# brew install python3
virtualenv -p /usr/local/bin/python3 venv
source venv/bin/activate
pip install numpy matplotlib jupyter pillow
jupyter notebook
```

`pillow` allows image processing for `jpg`.
